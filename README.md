<h1 align="center">💬 Real-Time Chat Application</h1>

<p align="center">
A real-time chat application built using <b>Node.js, Express, and Socket.io</b>.
Users can send and receive messages instantly using WebSocket communication.
</p>

<p align="center">
<b>🚀 Live Demo:</b><br>
<a href="https://chat-app-9w63.onrender.com/" target="_blank">
https://chat-app-9w63.onrender.com/
</a>
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
<li>Real-time messaging</li>
<li>Multiple users can join and chat simultaneously</li>
<li>Instant message broadcasting</li>
<li>Simple and responsive UI</li>
<li>WebSocket communication using Socket.io</li>
</ul>

<hr>

<h2>🛠️ Tech Stack</h2>

<ul>
<li><b>Node.js</b> – Backend runtime</li>
<li><b>Express.js</b> – Web server framework</li>
<li><b>Socket.io</b> – Real-time communication</li>
<li><b>HTML</b> – Chat interface structure</li>
<li><b>CSS</b> – Styling</li>
<li><b>JavaScript</b> – Client-side logic</li>
</ul>

<h2>⚙️ Installation & Setup</h2>

<h3>1. Clone the repository</h3>

<pre>
git clone https://github.com/Anubhav-Deepankar/ChatApp.git
</pre>

<h3>2. Navigate to the project directory</h3>

<pre>
cd ChatApp
</pre>

<h3>3. Install dependencies</h3>

<pre>
npm install
</pre>

<h3>4. Run the server</h3>

<pre>
node index.js
</pre>

<p>The server will start and the chat application will be available in your browser.</p>

<hr>

<h2>💡 How It Works</h2>

<ul>
<li>The server is built with <b>Node.js</b> and <b>Express</b>.</li>
<li><b>Socket.io</b> enables real-time WebSocket communication.</li>
<li>When a user sends a message, it is broadcast to all connected users instantly.</li>
</ul>
